# testTaskDS
Задача
Классификация товаров по их названиям.

Что сделано
    Обработка данных - сгруппировали категории в более крупные группы
    Обучение модели - использовали алгоритм логистической регрессии
    Преобразование текста - перевели названия товаров в числовые признаки
    Оценка качества - проверили, насколько хорошо модель предсказывает категории
Результаты
    Модель определяет категории товаров с точностью ~85-90%
    Лучше всего определяет популярные категории (электроника, одежда, книги)
    Сложнее с редкими и специфическими категориями
Что получили
    Обученную модель - готова к использованию на новых данных
    Файл submission.csv - предсказания для тестовых данных
    Метрики качества - понимание, насколько хорошо работает модель

Структура проекта:
    -data
        -raw  содержит в себе сырые данные (пример выходного файла, примеры для обучения модели, тестовые данные)
    -models  для сохранения обученных моделей
        -classifier_joblib  сохраненная модель классификатора
        -vectorizer_joblib  сохраненный векторайзер для текста
    - notebooks  Jupyter ноутбуки для анализа (отказадся от этой идеи)
    -src  исходный код проекта
        -data  содержит файлы для работы с данными
            -explore.py  исследование и анализ данных
            -preprocess.py  предобработка и очистка данных
        -models  код для моделей машинного обучения
            -baseline.py  базовая модель и ее реализация
    -submission.csv  файл с результатами для отправки

Файл explore.py для ознакомления с данными, preprocess.py обработка данных для создания супер категорий и анализа полученных категорий.
Файл baseline.py:
    create_baseline_model() 
        1. Загрузка данных - train_supervised_dataset.csv
        2. Предобработка - create_super_categories()
        3. Векторизация - TF-IDF с биграммами
        4. Обучение - Logistic Regression
        5. Оценка - F1-score на тестовой выборке
        6. Сохранение - Модели в папку models
        7. Предсказание - На test данных
        8. Создание submission - submission.csv
